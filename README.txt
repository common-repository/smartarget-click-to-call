Add click to call button on your site

== Description ==

Allow users to call you within a click!
If you offer customer support via phone calls, you must use Smartarget Click to Call. This app will add a phone icon on all your store pages and will allow your users to call you.
Making yourself available for your customers will give you the opportunity to sell more. Click to Call can be configured in your own way with the relevant message you would like to have.
Smartarget Click to Call plugin works also with WooCommerce, Elementor and Oxygen Builder.

== Installation ==

1. Unpack plugin zip to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Open plugin settings page
4. Configure settings and press Save
